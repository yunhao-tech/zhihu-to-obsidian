# v1.1 Product Iteration

## First Principles

Users do not pay because a page can be converted to Markdown.

They pay when the tool saves them from one of these expensive jobs:

- preserving high-value material before it disappears
- turning messy Chinese web pages into clean AI/RAG-ready Markdown
- keeping images and formulas usable after export
- batch archiving a source collection without repetitive manual work
- avoiding broken exports every time a platform changes DOM or API details

So the product should not be positioned as "a Zhihu exporter". It should be positioned as:

> A local-first Chinese knowledge archiver for Obsidian and AI workflows.

## v1.1 Scope

This release expands the extension from Zhihu-only to a broader Chinese article archiver.

### Supported in v1.1

- Zhihu articles
- Zhihu column batch export
- WeChat public account articles
- Juejin articles
- CSDN blog articles
- CNBlogs articles
- SSPAI articles
- Jianshu articles

### Still Differentiated

Zhihu remains the high-fidelity source:

- column detection
- batch column export
- range export
- formula handling
- article metadata
- image modes

Other platforms start with high-value single-page export. This is enough to test demand and collect failure cases.

## Paid Features Worth Testing

### Pro

- multi-platform export
- local image packaging
- Tailscale/NAS image paths
- Zhihu batch export
- export templates for Obsidian, RAG, and NotebookLM
- compatibility fixes for paid users

### Concierge

- archive one source collection end-to-end
- return an Obsidian-ready folder
- fix failed pages manually when needed

This is the fastest route to first revenue because it sells the result, not the extension.

## Why This Can Beat Free Clippers

Free generic clippers are broad but shallow. The paid wedge is platform-specific reliability:

- Chinese platforms use custom DOM and lazy image attributes.
- Images often need attachment packaging, not just remote URLs.
- Long-form users care about source metadata.
- AI workflow users care about clean Markdown more than page screenshots.
- Compatibility maintenance is ongoing work.

## Next Feature Bets

Prioritize after the first 10 beta conversations:

1. Selection export: export only highlighted content.
2. Markdown preview before download.
3. AI/RAG cleanup mode: remove ads, related posts, comment widgets, and navigation.
4. Export template presets: Obsidian, NotebookLM, RAG corpus.
5. WeChat image localizer hardening.
6. Batch URL list import.
7. Export history and failed-page retry report.
